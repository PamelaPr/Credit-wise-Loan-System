# Credit-wise-Loan-System

I developed a supervised machine learning pipeline to predict loan approval status.
The project involves - 
 1.Data preprocessing, 
 2.Exploratory data analysis (EDA) ,
 3.Feature Engineering,
 4.Training multiple classification models such as 
  -Logistic Regression, 
  -KNN, 
  -Naive Bayes
Since the problem was binary classification ,I evaluated model performance using
- precision ,
- recall,
- F1 score 
to ensure balanced performance rather than focusing on just accuracy.
Additionally, as the problem statement was - "high risk customers getting approved" - keeping that in mind I give priority to Precision score ,as higher Precision helps reduce  false positive cases.
